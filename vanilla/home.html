How the Algorithm Works

Canny combines four sequential steps to produce thin, clean, and robust edges — superior to simpler filters. Click Learn more on any step to see the math.

01 🔲

Grayscale & Gaussian Blur

The image is converted to grayscale — color is irrelevant for edge detection. A Gaussian filter then smooths the image to suppress noise that would create false edges.

Learn more

The Gaussian kernel assigns higher weight to central pixels. Noise is removed before gradient computation to avoid detecting noise as edges.

Kernel (3×3, divisor = 16): [[1, 2, 1], [2, 4, 2], [1, 2, 1]]
02 ↗

Sobel Gradient

Two Sobel kernels compute the image gradient in horizontal (Gx) and vertical (Gy) directions. Magnitude and direction are calculated for every pixel.

Learn more

G = √(Gx² + Gy²) measures edge strength. θ = atan2(Gy, Gx) gives the edge direction. Brighter output pixels indicate stronger edges.

Kx = [[-1,0,1],[-2,0,2],[-1,0,1]] Ky = [[ 1,2,1],[ 0,0,0],[-1,-2,-1]] G = √(Gx² + Gy²)
03 ✂

Non-Maximum Suppression

Only local maximum pixels along the gradient direction are kept. All others are suppressed, thinning wide edge bands into single-pixel-wide lines.

Learn more

For each pixel the two neighbors in its gradient direction are checked. If the pixel is not the local maximum it is set to zero — eliminating blurry, thick edges.

If G(x,y) < G(neighbor₁) OR G(x,y) < G(neighbor₂): → suppress pixel (set to 0)
04 🎯

Hysteresis Thresholding

Two thresholds classify pixels as strong edges, weak edges, or non-edges. Weak edges connected to strong edges are kept; isolated weak edges are discarded.

Learn more

This removes noise while preserving real edge continuity. You can adjust these thresholds interactively on the Convert page.

I > T_high → strong edge ✓ I < T_low → non-edge ✗ T_low ≤ I ≤ T_high → weak edge (kept only if adjacent to strong edge)
